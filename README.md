# ASEGURA - Consultora de Seguridad Empresarial

Sitio web corporativo para ASEGURA, consultora especializada en seguridad empresarial, auditoría, prevención de riesgos y cumplimiento normativo en Perú.

## 🌐 Enlace del Proyecto

El proyecto se encuentra en: `C:/Users/marketing/Desktop/asecura/`

## 🚀 Cómo Ejecutar

### Opción 1: Ver directamente (sin npm)
El proyecto ya está compilado en la carpeta `build/`. Abra directamente el archivo:

```
build/index.html
```

### Opción 2: Desarrollo con Gulp (requiere Node.js)

```bash
# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm start

# Solo compilar
npm run build
```
